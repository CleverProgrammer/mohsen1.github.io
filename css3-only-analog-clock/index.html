CSS3 Only Analog Clock

Feb 21st, 2012

This is an analog clock that uses zero images and no JavaScript for animation. JavaScript is used only to get current time from browser Date and Time API. This clock is first clock that have haptic feedback animation second hand. Lunch the experiment
